## Vamos começar a ultilizar nossa calculadora
Para exercutar o arquivo .sh voce necessita usar o comando ./Minha_calculadora.sh para executar o script no terminal."

## Funções da calculadora em python:
 -Adição
 -Subtração
 -Multiplicação
 -Divisão

## Como Ultilizar:

Você deve inserir o primeiro numero desejado, tanto em formato INT(inteiro) ou FLOAT(fracionado), depois inserir o simbolo da operação e depois o segundo número.
Se a divisão for feita por 0, a calculadora retornará que não é possivel dividir por 0.
Depois de feito a operação, a calculadora mostrará se o número é impar ou par.
